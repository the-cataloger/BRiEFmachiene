# BRIEF Machine

**BRIEF Machine** هو تطبيق بسيط يعتمد على Python وواجهة Tkinter الرسومية لتحويل صور (JPG، JPEG، PNG) من مجلدات فرعية إلى ملفات PDF. يقوم البرنامج بمعالجة كل مجلد فرعي في دليل محدد، حيث يختار عددًا محددًا من الصور (افتراضيًا: أول 3 وآخر 3 صور) أو جميع الصور إذا كان عددها أقل من العدد المطلوب، ثم يحولها إلى ملف PDF.

## المميزات
- واجهة رسومية سهلة الاستخدام لاختيار المجلدات.
- دعم تنسيقات الصور الشائعة (JPG، JPEG، PNG).
- ترتيب الصور بشكل طبيعي باستخدام مكتبة `natsort`.
- تنظيف الملفات المؤقتة تلقائيًا بعد إنشاء PDF.
- سجل الأخطاء والتقدم لتتبع العمليات.
- مرونة في تحديد عدد الصور لكل ملف PDF.

## المتطلبات
- Python 3.6 أو أحدث.
- المكتبات المطلوبة:
  - Pillow (`pip install Pillow`)
  - PyPDF2 (`pip install PyPDF2`)
  - natsort (`pip install natsort`)
- نظام تشغيل متوافق مع Tkinter (Windows، macOS، Linux مع إعداد Tkinter).

## التثبيت
1. قم بتثبيت Python من [python.org](https://www.python.org).
2. قم بتثبيت المكتبات المطلوبة:
   ```bash
   pip install Pillow PyPDF2 natsort
   ```
3. قم بتنزيل ملف `image_to_pdf.py` من المستودع.
4. (اختياري) إذا كنت تستخدم مجلد شبكة، تأكد من وجود صلاحيات الوصول.

## كيفية الاستخدام
1. قم بتشغيل البرنامج:
   ```bash
   python image_to_pdf.py
   ```
2. في الواجهة الرسومية:
   - انقر على "استعراض" لاختيار المجلد الرئيسي الذي يحتوي على المجلدات الفرعية بالصور.
   - انقر على "استعراض" لاختيار مجلد الوجهة لحفظ ملفات PDF.
   - أدخل عدد الصور لكل PDF (الافتراضي: 3 من البداية و3 من النهاية).
   - انقر على "إنشاء ملفات PDF" لبدء العملية.
3. تحقق من ملفات PDF الناتجة في مجلد الوجهة.

## ملاحظات
- يتم حفظ ملف PDF لكل مجلد فرعي باسم المجلد.
- إذا كان عدد الصور في المجلد الفرعي أقل من العدد المحدد، سيتم استخدام جميع الصور.
- يتم إنشاء سجل للعمليات في وحدة التحكم لتتبع الأخطاء أو التحذيرات.
